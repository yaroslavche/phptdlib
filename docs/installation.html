Installation

Note

The most of dependencies are installed via git submodules currently. But PHP-CPP currently has to be built separately.

CopernicaMarketingSoftware/PHP-CPP

git clone https://github.com/CopernicaMarketingSoftware/PHP-CPP.git
cd PHP-CPP
make
sudo make install

yaroslavche/phptdlib

Note

If you want to link other dependencies as dynamic libraries, you can pass some options to cmake. Look through CMakeLists.txt comments to find out how. Required more than 4GB RAM and must wait a while. 

git clone --recurse-submodules https://github.com/yaroslavche/phptdlib.git
cd phptdlib && mkdir build && cd build
cmake ..
make
sudo make install

Full installation example on Ubuntu 18.04

Go to some dir where phptdlib and PHP-CPP dirs will be placed: 

$ cd ~/projects
Install required dependencies: 

$ sudo apt install gperf zlib1g-dev php-all-dev
Clone and build PHP-CPP: 

$ git clone https://github.com/CopernicaMarketingSoftware/PHP-CPP.git
$ cd PHP-CPP
$ make
$ sudo make install
Return back into installation dir: 

$ cd ../
Clone and build phptdlib: 

$ git clone --recurse-submodules https://github.com/yaroslavche/phptdlib.git
$ cd phptdlib && mkdir build && cd build
$ cmake ..
$ make
$ sudo make install
And check if it works: 

$ php -m | grep tdlib
$ php ../php_examples/func.php
$ php -r "echo (function_exists('td_json_client_create') ? 'ok' : 'failed') . PHP_EOL;"
